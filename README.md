# Compilador - v2.1
Simple Calculator - PHP
---
## Run command example : 

```python
$ python main.py arquivo.php
```
Exemplo de arquivo.php
```python
{
   $a1=12;
   $a2=24;
   echo 1+2*($a2/$a1);
}
```
---
## Supported ops 
```python
tokens=["+","-","*","/","(",")","Number"]
```
